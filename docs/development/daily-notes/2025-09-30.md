### üèóÔ∏è Technical Architecture

**Movement Algorithm**:
```javascript
// Organic elliptical paths using sin/cos with phase offsets
targetX = centerX + sin(time * speed + phaseX) * amplitude
targetY = centerY + cos(time * speed * 0.7 + phaseY) * amplitude * 0.8

// Different frequencies (1.0 vs 0.7) create lissajous curves
// Asymmetric amplitudes (1.0 vs 0.8) add variety
```

**Mouse Attraction**:
```javascript
// Calculate distance to cursor
distance = sqrt((mouseX - targetX)¬≤ + (mouseY - targetY)¬≤)

// Apply influence based on proximity
if (distance < 300px) {
  factor = (1 - distance/300) * mouseInfluence
  targetX += dx * factor
  targetY += dy * factor
}

// Smooth with lerp to avoid jitter
orb.x = lerp(orb.x, targetX, 0.05)
```

**Gradient Rendering**:
```javascript
// Radial gradient with soft edges
gradient = createRadialGradient(x, y, 0, x, y, radius)
gradient.addColorStop(0, rgba(r,g,b, 0.25))    // Center: bright
gradient.addColorStop(0.4, rgba(r,g,b, 0.15))  // Mid: medium
gradient.addColorStop(0.7, rgba(r,g,b, 0.075)) // Edge: dim
gradient.addColorStop(1, rgba(r,g,b, 0))       // Fade to transparent

// Additive blending for luminous overlaps
ctx.globalCompositeOperation = 'lighter'
```
